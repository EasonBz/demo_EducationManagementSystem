
## 后端  
    1. IDEA 打开项目，自动下载依赖；  
    2. IDEA 打开 File -> Setting -> Build, Execution, Deployment -> Build Tools -> Maven;  
	Maven home directory 选项，导入 maven 文件;  
## 前端 
    开发环境：  
        进入 前端目录 sms, 执行;  
        1. npm install  下载依赖;  
        2. npm run dev   启动;  
        3. 访问 http://localhost:8443/
## 数据库：  
     student_mis.sql, 直接导入Mysql;  
     在 application.properties 文件下更改数据库账号密码;
